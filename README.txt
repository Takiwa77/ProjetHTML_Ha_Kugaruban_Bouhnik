Ouvrir le site :
1. Aller dans le dossier Html
2. Ouvrir accueil.html ou index.html selon la page d'accueil présente.

Structure :
- Html : toutes les pages HTML
- css : feuilles de style
- js : scripts JavaScript
- assets : images / données nécessaires au site

Les liens CSS/JS ont été corrigés pour fonctionner avec cette organisation.
